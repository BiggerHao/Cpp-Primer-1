Exercise 2.25: Determine the types and values of each of the following
variables.

	(a) int* ip, i, &r = i;		(b) int i, *ip = 0;		(c) int* ip, ip2;

By Faisal Saadatmand

(a)
	ip a pointer to int; uninitialized
	i is an int; uninitialized
	r is reference to an int, initialized with i
(b)
	i is an int; uninitialized
	ip is a pointer to an int; initialized to null or 0
(c)
	ip is a pointer to int; uninitialized
	ip2 is an int; uninitialized
